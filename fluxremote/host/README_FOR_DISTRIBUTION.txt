FluxRemote Host Package
=======================

This archive contains the host-side Python app for FluxRemote.

How to run on another Windows machine:
1. Install Python 3.10+.
2. Open Command Prompt in this folder.
3. Run: pip install -r requirements.txt
4. Run the host with your server URL, device ID, and password:
   python main.py --server ws://YOUR_SERVER_URL --id YOUR_DEVICE_ID --password YOUR_PASSWORD

If you want, I can also help build a standalone .exe for easier distribution.
